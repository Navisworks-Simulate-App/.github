## 🚀 Introduction
Navisworks Simulate for Windows is Autodesk software built for whole-project review and coordination. It answers a common question on projects — what is Navisworks Simulate for — by aggregating design and construction models from many applications into a single scene you can navigate, measure, and mark up. Beyond static review, it links model geometry to a project schedule through TimeLiner, so you can play the build order as a 4D simulation and animate objects to demonstrate phasing and logistics. The result is a clear, shared picture of how a project fits together and how it will be constructed, useful for architects, engineers, planners, and contractors alike.

## ✨ Highlights

### Review and Coordination
- **Multi-format aggregation** — Bring models from many CAD and BIM sources into one coordinated project without altering the originals.
- **Smooth navigation** — Walk, fly, and orbit through large aggregated models, then section and measure to inspect details.
- **Markups and viewpoints** — Redline, comment, and save camera views to record and share review decisions.

### Simulation and Animation
- **TimeLiner 4D scheduling** — Connect model elements to task dates and simulate the construction sequence in time.
- **Object animation** — Script movement of equipment and elements to visualize phasing, installation, and site activity.

## 🎯 Getting Up and Running
1. Use **Append** to load all your model files into a single aggregated scene.
2. Navigate the coordinated model with the walk, orbit, and examine tools to review the design.
3. Open TimeLiner, build or import a task list, and link model selections to each task.
4. Set dates and simulation settings, then play the sequence to watch construction progress in 4D.
5. Save viewpoints and export animations to share the review across your team.

## ❓ Common Questions
**What is Navisworks Simulate?**
Navisworks Simulate is an Autodesk project review application that aggregates multi-format models, lets you review and navigate them in 3D, animates objects, and builds 4D construction simulations using TimeLiner scheduling. It is designed to help teams coordinate a project and communicate how it will be built.

**Is Navisworks Simulate free?**
No — it is a paid Autodesk product available with a free trial for evaluation. If you only need to view published models, Autodesk offers Navisworks Freedom as a free viewer for NWD and DWF files. Freedom is free but omits the simulation and animation features that Simulate provides.

**How does the Navisworks Simulate interface work?**
The interface centers on a large 3D review window surrounded by dockable panels — a selection tree for model contents, plus panels for viewpoints, TimeLiner scheduling, animation, and markups. Ribbon tabs group navigation, review, and simulation tools, so you can move from inspecting a model to building a 4D simulation within one workspace.

**Which versions of Windows are supported?**
Navisworks Simulate runs on 64-bit Windows, including current, supported releases such as Windows 10 and Windows 11.
